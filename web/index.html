Oklit Blogging platform for developpers

Oklit is an open-source blogging platform made for developpers, with it you can create a successfull weblog and customize it to suite your needs.

Features

Code coloration

Use the github syntax to colorize code snippets in any language

Manage your posts/pages

The admin panel allows you to manage posts and pages, keyboard navigation available

Write your posts in markdown

With markdown you can keep your post readable and render it in HTML.

Comments

Comment moderation, gravatar images, markdown syntax..

Backup your posts and images

Create an archive containing a dump of your database and the /upload folder

Filesystem manager

Upload somes file to a web directory, add images for your posts..

Installation

Clone the project:

git clone --recursive git@github.com:gordonslondon/Oklit.git && cd Oklit

Now copy and fill app/config.php with your database user and password:

cp app/config.php.dist app/config.php && vi app/config.php

Your done! now go to your browser and see the result.

Access backend

Create the database:

mysqladmin -u admin -p create silex_blog_dev

Load an empty schema:

mysql -u admin -p silex_blog_dev < app/schema.sql

Create a new user:

php app/console.php user -u USER -p PASS -m admin@mail.com

Your done! now go to /admin

Create a theme

Themes allow you to keep your modifications in a self-contained folder, it's the easiest way to keep your blog upgradable.

Theming support allow your to override or create:

  • twig templates
  • css and images files
  • controllers

To create a new theme:

create a directory under web/themes/YOUR_THEME_NAME.

enable it in the app/config.php file, set the value of the theme key to YOUR_THEME_NAME.

Your done! you can start working on your theme.
